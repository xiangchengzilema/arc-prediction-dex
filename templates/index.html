How Pythia thinks

An AI trader, fully on-chain

Pythia runs a tight loop on Arc. Every 30 seconds it surveys every market, prices probabilities, and acts only when expected edge clears its threshold.

01 / SCAN

Read the AMM

Every cycle, Pythia pulls live YES/NO prices from the on-chain AMM and turns them into implied probabilities.

02 / PRICE

Estimate true odds

A probability model produces a target price. Edge is the gap between target and AMM. No edge above 3%, no trade.

03 / SIZE

Kelly-fractional sizing

Position size = ¼ Kelly. It maximises long-run growth without going bust on a string of bad reads.

04 / EXECUTE

Settle on Arc

Trades clear in under a second on Arc with USDC-native rails — fees around one cent each, fast enough to react.

05 / OBSERVE

Transparent decisions

Every action — buy, hold, skip — is logged with full reasoning. Watch the agent think in real time.

06 / LEARN

Track and adjust

Realised and unrealised P&L flow into the leaderboard. Strategy parameters can be tuned from the agent control room.
